# Space-Punk

2D physics puzzle game built with Python and Pygame.

## Run

1. Install dependencies:
   - `pip install -r requirements.txt`
2. Launch:
   - `python main.py`

## Controls

- Aim slingshot: mouse drag
- Launch planet/moon: mouse release
- Restart current level: `R`
- Pause: `ESC`
- Pause menu to main menu: `M`

## Included Levels

- Level 1: Normal Space
- Level 2: Meteor Attack
- Level 3: Asteroid Field (Moon Phase)
- Level 4: Black Hole Galaxy Transfer
